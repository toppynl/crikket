Use the Capture SDK to add the Crikket bug-report widget directly to your site.

## In This Section

- [Quick Start](/docs/usage/quick-start)
- [Integration](/docs/usage/integration)

## What You Need

- A Crikket workspace
- An org admin or owner who can create public keys
- A website or app origin you control, such as:
  - `https://example.com`
  - `https://www.example.com`
  - `http://localhost:3000`

## How It Works

1. Create a site-scoped public key in Crikket.
2. Add the exact allowed origins where the widget can run.
3. Initialize `@crikket/capture` in your site.
4. Users launch the widget, capture a screenshot or screen recording, and submit the report from your app.
